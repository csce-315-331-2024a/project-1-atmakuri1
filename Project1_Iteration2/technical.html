CSCE 121: Dungeon Crawler

I developed a text-based game using C++, focusing on seamless code integration and efficient. I engineered complex game mechanics, including character movement, collision detection, and treasure collection, utilizing the Standard Template Library (STL). Employing object-oriented design patterns, I optimized the code structure for modularity. Additionally, I incorporated the C++ File I/O library to enable the smooth loading of dungeon levels from external files, enhancing the game's flexibility. Proficient in C++ programming, I showcased expertise in dynamic memory management, polymorphism, and encapsulation. The project also involved the design and creation of a captivating user interface, enhancing the overall gaming experience for players.

CSCE 121: Seam Carving

This was a C++ project and I used essential libraries including iostream, string, sstream, fstream, and cmath to implement dynamic memory management for creating, manipulating, and saving images in the P3 format. Leveraging advanced algorithms, I designed content-aware seam carving techniques to precisely identify and remove minimal energy seams, allowing for sophisticated image resizing and content-aware transformations. The project showcases not only technical expertise but also meticulous attention to detail, demonstrated through comprehensive error handling strategies that enhance the robustness and reliability of the entire image processing workflow.
